# 🚀 Nemeis Bot 

O **Bot Nemesis** é um sistema inteligente de automação para servidores Discord, desenvolvido em **Python** utilizando a biblioteca `discord.py`.

O projeto foi estruturado com foco em organização, escalabilidade e boas práticas de desenvolvimento back-end, simulando um ambiente real de produção.

---

##  Objetivo

Desenvolver uma solução automatizada capaz de:

-  Gerenciar ações internas de forma estruturada  
-  Monitorar e registrar lives automaticamente  
-  Controlar registros de usuários com botões interativos  
-  Aplicar limite de uso semanal com reset automático  

---

## Diferenciais Técnicos

- Arquitetura modular (separação de responsabilidades)
- Persistência de dados utilizando arquivos JSON
- Programação assíncrona com `asyncio`
- Automação com `tasks.loop`
- Componentes interativos do Discord (Views e Buttons)
- Estrutura preparada para deploy (Discloud, VPS ou similares)

---

## Tecnologias Utilizadas

- Python 3.10+
- discord.py
- AsyncIO
- JSON
